    **Per-day loop logic (in `run_hybrid_backtest`):**

    1. `t0 = time.time()`
    2. Load all OHLCV + VIX once (same pattern as quant_backtest.py): `all_rows`, `all_vix`, `row_by_date`, `vix_by_date`, `all_dates`, `vix_dates`
    3. Compute `date_range = [d for d in all_dates if from_date <= d <= to_date]`
    4. For each `date` in `date_range`:

       a. Build `closes_up_to`, `rows_up_to` (no future leakage — filter `<= date`)

       b. Compute `rsi`, `supertrend`, `vix`, `vix_5d_avg`, `momentum_5d` (same pattern as quant_backtest.py lines 112-120)

       c. Build `QuantInputs` (same proxy formula as existing code)

       d. `quant_result = QuantSignalEngine.compute_quant_score(inputs, instrument)`

       e. Build `MarketInput` (same pattern as hybrid route in routes.py lines 1229-1237)

       f. Dispatch agents: `sim_result = await orchestrator.run_simulation(market_input)` — Orchestrator already runs 6 agents in parallel (Phase 11 rewrite)
          - `final_outputs = sim_result["final_outputs"]`
          - Wrap in try/except; on exception: log warning and use empty dict for final_outputs

       g. Build placeholder verdict and call `HybridScorer.fuse` once for LLMValidator prompt:
          ```python
          placeholder = ValidatorVerdict(verdict="confirm", reasoning="", adjustment_amount=0.0)
          pre_hybrid = HybridScorer.fuse(quant_result, final_outputs, placeholder)
          ```

       h. Call validator with try/except fallback to confirm:
          ```python
          try:
              verdict = await LLMValidator.call_validator(pre_hybrid, instrument, date)
          except Exception:
              verdict = ValidatorVerdict(verdict="confirm", reasoning="Validator unavailable.", adjustment_amount=0.0)
          ```

       i. Final fuse: `hybrid = HybridScorer.fuse(quant_result, final_outputs, verdict)`

       j. Compute `actual_move_pct` (None for last day — same pattern as quant_backtest.py)

       k. Compute `is_correct`:
          - `None` when `hybrid.direction == "HOLD"` or `hybrid.tier == "skip"` or `actual_move_pct is None`
          - `True` when BUY and actual_move_pct > 0
          - `True` when SELL and actual_move_pct < 0
          - Otherwise `False`

       l. Compute risk-adjusted P&L:
          ```python
          pnl_points = 0.0
          lots = 0
          if hybrid.tradeable and hybrid.direction != "HOLD" and actual_move_pct is not None:
              entry_close = closes_up_to[-1]
              vix_for_risk = vix if vix > 0 else 15.0
              rp = RiskManager.compute(hybrid.tier, hybrid.direction, entry_close, vix_for_risk)
              lots = rp.lots
              actual_move_pts = (actual_move_pct / 100.0) * entry_close
              if hybrid.direction == "BUY":
                  pnl_points = actual_move_pts * lots * self.NIFTY_LOT
              elif hybrid.direction == "SELL":
                  pnl_points = -actual_move_pts * lots * self.NIFTY_LOT
          ```

       m. Append `HybridBacktestDayResult(...)` to results list

    5. After loop, compute aggregates (same formulas as Plan 01 Task 1):
       - `tradeable`: days where `is_correct is not None`
       - `correct`: days where `is_correct == True`
       - `win_rate_pct`, `total_pnl_points`
       - Sharpe, max_drawdown_pct, win_loss_ratio (use `statistics` stdlib, same approach as Plan 01)

    6. Return `HybridBacktestRunResult(...)` with `elapsed_seconds=time.time()-t0`

    @protected_router.post("/backtest/hybrid-run", response_model=HybridBacktestRunResponse)
    async def run_hybrid_backtest(body: HybridBacktestRequest):
        """Run a hybrid backtest (quant + agents + validator + risk) over a date range.

        Each day runs: QuantSignalEngine + 6 parallel agents + HybridScorer + LLMValidator + RiskManager.
        Performance target: 1 month (~22 trading days) in under 5 minutes (FBT-02).

        Returns per-day hybrid results and aggregate risk-adjusted metrics.
        """
        if body.instrument not in ("NIFTY", "BANKNIFTY"):
            raise HTTPException(status_code=400, detail="instrument must be NIFTY or BANKNIFTY")

        try:
            result = await hybrid_backtest_engine.run_hybrid_backtest(
                body.instrument, body.from_date, body.to_date
            )
        except ValueError as exc:
            raise HTTPException(status_code=400, detail=str(exc))
        except Exception as exc:
            logger.exception("Hybrid backtest failed")
            raise HTTPException(status_code=500, detail=f"Hybrid backtest failed: {exc}")

        day_schemas = [
            HybridBacktestDaySchema(

                is_correct=d.is_correct,
                pnl_points=d.pnl_points,
