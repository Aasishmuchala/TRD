POST /api/signal/hybrid/{instrument}/{date} returns HybridSignalResponse:
```
{
  direction: "BUY" | "SELL" | "HOLD",
  hybrid_score: number,          // 0-100
  conviction: number,            // 0-100 (post-validation)
  tradeable: boolean,
  tier: "strong" | "moderate" | "skip",
  instrument_hint: string,       // e.g. "NIFTY CE"
  quant_breakdown: {
    total_score: number,
    direction: string,
    factors: {
      [factor_key]: {
        threshold_hit: boolean,
        side: "buy" | "sell",
        points_awarded: number
      }
    }
  },
  agent_breakdown: [
    {
      agent_id: string,
      direction: "BUY" | "SELL" | "HOLD",
      conviction: number,
      rationale: string
    }
  ],
  validator_verdict: "confirm" | "adjust" | "skip",
  validator_reasoning: string,
  risk_params: {
    position_size_lots: number,
    stop_loss_pts: number,
    target_pts: number,
    stop_level: number,
    target_level: number,
    vix_used: number
  }
}
```

GET /api/signal/quant/{instrument}/{date} returns QuantSignalResponse:
```
{
  total_score: number,
  direction: "BUY" | "SELL" | "HOLD",
  tier: string,
  factors: { [key]: { threshold_hit, side, points_awarded } }
}
```
