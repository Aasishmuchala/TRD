  <files>gods-eye/backend/app/engine/signal_scorer.py, gods-eye/backend/tests/test_signal_scorer.py</files>
  <behavior>
    Scoring formula (from CONTEXT.md decisions):

    sentiment_component (60% weight):
      - Map direction to base multiplier: STRONG_BUY/STRONG_SELL → 1.0, BUY/SELL → 0.85, HOLD → 0.0
      - sentiment_raw = conviction * direction_multiplier  (0-100)
      - sentiment_score = sentiment_raw * 0.60

    technical_component (40% weight):
      - RSI alignment: rsi < 30 supports BUY; rsi > 70 supports SELL; otherwise neutral
      - Supertrend alignment: "bullish" supports BUY; "bearish" supports SELL
      - VIX regime alignment: "low" supports BUY (complacency → reversal risk acknowledged; per CONTEXT: low supports BUY); "high" supports SELL; "normal"/"elevated" neutral
      - OI sentiment alignment: "bullish" supports BUY; "bearish" supports SELL; absent → neutral
      - Count aligned technicals (0-4), each worth 25 points → tech_raw = aligned_count * 25
      - technical_score = tech_raw * 0.40

    final_score = round(sentiment_score + technical_score, 1)  — clamped to [0, 100]

    HOLD direction: sentiment_component = 0 regardless of conviction, max score = 40
    (technicals can still score but no sentiment, so HOLD always < 50 when <3 technicals align)

    Tier mapping:
      score > 70  → tier = "strong"
      score >= 50 → tier = "moderate"
      score < 50  → tier = "skip"

    Suggested instrument (based on direction + instrument param):
      - BUY/STRONG_BUY + NIFTY    → "NIFTY_CE"
      - SELL/STRONG_SELL + NIFTY  → "NIFTY_PE"
      - BUY/STRONG_BUY + BANKNIFTY → "BANKNIFTY_CE"
      - SELL/STRONG_SELL + BANKNIFTY → "BANKNIFTY_PE"
      - HOLD (any)                → "NONE"

    contributing_factors list — include only factors that actually aligned:
      Examples: "RSI oversold (28.3) supports BUY", "Supertrend bullish", "VIX low regime supports BUY"
      Always include: "Agent conviction: 72.0 (BUY)"

    Test cases (RED phase must fail, GREEN must pass):
      1. score(direction="BUY", conviction=80, signals={"rsi": 25, "supertrend": "bullish",
         "vix_regime": "low", "oi_sentiment": "bullish"}, instrument="NIFTY")
         → score > 70, tier = "strong", suggested_instrument = "NIFTY_CE"

      2. score(direction="HOLD", conviction=90, signals={"rsi": 25, "supertrend": "bullish",
         "vix_regime": "low", "oi_sentiment": "bullish"}, instrument="NIFTY")
         → score < 50, tier = "skip", suggested_instrument = "NONE"

      3. score(direction="SELL", conviction=60, signals={"rsi": 50, "supertrend": "bullish",
         "vix_regime": "normal"}, instrument="BANKNIFTY")
         → 0 technicals aligned (rsi neutral, supertrend bullish opposes SELL, vix normal)
         → sentiment_score = 60 * 0.85 * 0.60 = 30.6, technical_score = 0
         → score = 30.6, tier = "skip", suggested_instrument = "BANKNIFTY_PE"

      4. score(direction="BUY", conviction=55, signals={"rsi": 50, "supertrend": "bullish",
         "vix_regime": "normal"}, instrument="NIFTY")
         → 1 technical aligned (supertrend only)
         → sentiment = 55 * 0.85 * 0.60 = 28.05, technical = 25 * 0.40 = 10.0
         → score ≈ 38.1, tier = "skip"

      5. score() returns a ScoreResult dataclass with fields:
         score: float, tier: str, direction: str, contributing_factors: List[str],
         suggested_instrument: str

      6. score() clamps final value: score never < 0, never > 100

      7. STRONG_BUY with full technical alignment:
         conviction=100, direction="STRONG_BUY", all 4 aligned
         → sentiment = 100 * 1.0 * 0.60 = 60, technical = 100 * 0.40 = 40
         → score = 100.0, tier = "strong"
  </behavior>
