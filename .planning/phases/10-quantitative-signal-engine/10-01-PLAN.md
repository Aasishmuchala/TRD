```python
@dataclass
class QuantInputs:
    # All fields required for scoring. Plan 02 builds this from historical data.
    fii_net_cr: float          # FII net flow in crores (positive = buying)
    dii_net_cr: float          # DII net flow in crores (positive = buying)
    pcr: float                 # Put/Call ratio (>1.2 bearish sentiment = contrarian bullish)
    rsi: float                 # RSI-14 value (0-100)
    vix: float                 # Current India VIX level
    vix_5d_avg: float          # 5-day average VIX for direction detection
    supertrend: str            # "bullish" or "bearish"

@dataclass
class QuantScoreResult:
    total_score: int           # 0-100 clamped
    direction: str             # "BUY" | "SELL" | "HOLD"
    buy_points: int            # Raw buy points accumulated
    sell_points: int           # Raw sell points accumulated
    factors: Dict[str, Dict]   # Per-factor breakdown {name: {points, threshold_hit, side}}
    tier: str                  # "strong" (>70) | "moderate" (50-70) | "skip" (<50)
    instrument_hint: str       # "NIFTY_CE" | "NIFTY_PE" | "BANKNIFTY_CE" | "BANKNIFTY_PE" | "NONE"

class QuantSignalEngine:
    @staticmethod
    def compute_quant_score(inputs: QuantInputs, instrument: str = "NIFTY") -> QuantScoreResult:
        # Pure function. No I/O.
```

  <behavior>
    Scoring rules (exact thresholds from CONTEXT.md):

    BUY-side accumulators:
    - fii_buy: fii_net_cr > 1000  → +25 pts  (factor name: "fii_flow")
    - dii_absorption: dii_net_cr > 0 AND fii_net_cr < 0 → +10 pts  (factor name: "dii_absorption")
    - pcr_bullish: pcr > 1.2  → +15 pts  (factor name: "pcr")
    - rsi_oversold: rsi < 30  → +20 pts  (factor name: "rsi")
    - vix_calm: vix < 14 AND vix < vix_5d_avg  → +10 pts  (factor name: "vix")
    - supertrend_bull: supertrend == "bullish"  → +10 pts  (factor name: "supertrend")

    SELL-side accumulators:
    - fii_sell: fii_net_cr < -1000  → +25 pts  (factor name: "fii_flow", side=sell)
    - pcr_bearish: pcr < 0.7  → +15 pts  (factor name: "pcr", side=sell)
    - rsi_overbought: rsi > 70  → +20 pts  (factor name: "rsi", side=sell)
    - vix_fear: vix > 20 AND vix > vix_5d_avg  → +10 pts  (factor name: "vix", side=sell)
    - supertrend_bear: supertrend == "bearish"  → +10 pts  (factor name: "supertrend", side=sell)

    Direction logic:
    - if buy_points > sell_points AND buy_points > 50 → direction = "BUY"
    - if sell_points > buy_points AND sell_points > 50 → direction = "SELL"
    - else → direction = "HOLD"

    total_score = max(buy_points, sell_points) clamped to [0, 100]

    Tier: total_score > 70 → "strong", >= 50 → "moderate", < 50 → "skip"

    Instrument hint: BUY + NIFTY → "NIFTY_CE", BUY + BANKNIFTY → "BANKNIFTY_CE",
                     SELL + NIFTY → "NIFTY_PE", SELL + BANKNIFTY → "BANKNIFTY_PE",
                     HOLD → "NONE"

    Test cases:
    - All BUY signals active (fii=+2000, dii=+500 with fii negative — so dii_absorption: not triggered
      when fii is positive; use fii=-500, dii=+800 to test dii_absorption separately)
    - fii_net=+2000, pcr=1.5, rsi=25, vix=12/vix5d=13, supertrend=bullish → buy_points=80, direction=BUY, tier=strong
    - fii_net=-2000, pcr=0.5, rsi=75, vix=25/vix5d=22, supertrend=bearish → sell_points=80, direction=SELL, tier=strong
    - fii_net=0, pcr=1.0, rsi=50, vix=18/vix5d=18, supertrend=bullish → buy_points=10, sell_points=0, direction=HOLD (10 < 50), tier=skip
    - dii absorption: fii_net=-500, dii_net=+800, all others neutral → buy_points=10, direction=HOLD
    - buy exactly 50: arrange buy_points=50 → direction=HOLD (not strictly > 50)
    - buy exactly 51: arrange buy_points=51 → direction=BUY
    - instrument=BANKNIFTY + BUY → instrument_hint=BANKNIFTY_CE
    - instrument=BANKNIFTY + SELL → instrument_hint=BANKNIFTY_PE
    - HOLD always → instrument_hint=NONE
    - factors dict contains all evaluated factors with points and threshold_hit bool
